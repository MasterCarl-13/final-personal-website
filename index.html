My Project

Low-Cost Earthquake Detection Alarm System (LCEDAS)

The LCEDAS is a cost-effective earthquake detection and alarm system developed using Arduino technology and accelerometer sensors. Designed to address the need for accessible earthquake warning systems in the Philippines, it enhances community preparedness and safety in earthquake-prone areas. The system effectively detects seismic activity and provides timely alerts, demonstrating high functionality and efficiency.
